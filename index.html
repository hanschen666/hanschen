URL Bookmarks

This is a Google Chrome Extension project I created. It really solves my daily headaches. I often get frustrated because I have too many pages open in my browser and I can't close them because I need to use them. So I came up with a Google Chrome Extension to solve this problem, which is an extension that can store urls and then allow people to continue using them. When people have too many urls open in their browser, they can optionally use this extension to store urls and then open them at any time. I think that's the joy of technology, solving life's little problems.
